# 📄resume-portfolio

HTML, CSS로 제작한 개인 포트폴리오 입니다.  
프론트엔드 기초 학습 내용을 실제 화면 구성으로 정리했습니다.

## 🛠 기술 스텍

- HTML
- CSS
- Git Hub

## ✨ 주요 기능

- 반응형 레이아웃 (모바일 / 태블릿 / PC 대응)
- 연습 버튼 클릭 시 오버레이(모달) 표시
- CSS 가상 요소를 활용한 UI 디테일

## 🧠 문제 해결 경험

- 오버레이 레이아웃에서 정렬 문제를 겪어
  flex속성과 Position 조합을 통해 정렬 문제 해결
- 반응형 대응이 힘들었지만 교안페이지를 보고 대응

## 📚 느낀점

1. 기초 기술을 실제 화면으로 구현하며 HTML, CSS 구조에 대한 이해를 조금 더 높일 수 있었습니다.
2. 부족한 점을 인식하고 있으며, 반복적인 구현과 학습을 통해 성장하고자 합니다.
