About Me

자동차정비 전공으로 문제의 원인을 구조적으로 분석하는 역량을 쌓았으며, 이를 바탕으로 논리적인 코드 작성과 디버깅에 강점을 가진 프론트엔드 개발자입니다. ESTsoft 부트캠프를 통해 웹 개발 기초를 다지며, 사용자 경험을 고려한 직관적인 UI 구현에 집중하고 있습니다.

경험

웹사이트 클론 제작 연습

SSG, NETFLEX 등 실제 서비스 UI를 분석하고 일부 페이지를 클론 코딩했습니다. 클론코딩 과정에서 구조 파악, 컴포넌트화, 재사용 가능한 CSS 구조 설계를 연습하며 효율적인 코드 작성 방법을 연습했습니다. 또한 시멘틱 태그를 적극 활용하여 코드의 가독성과 유지보수성을 높이는 방법을 학습했습니다.

부트캠프 기초 과제 수행

ESTsoft 프론트엔드 부트캠프를 참여하면서 HTML, CSS 이론을 마쳤고 Flex와 Grid를 활용하여 카드 UI 구성 및 레이아웃 정리를 수행하였습니다. 현재는 개인 프로젝트로 이력서 페이지 제작을 하고 있으며 실제 화면 개발 경험을 확장하고 있습니다. 또는 모바일, 태블릿, PC에 따른 반응형 CSS을 구성함으로서 화면 구성 능력을 기르고 있습니다.
