프로필

자동차정비를 전공하며 쌓은 체계적인 문제 해결 능력과 복잡한 시스템 구조를 분석하는 역량은 웹 개발에서도 큰 강점으로 작용한다고 생각합니다. 차량의 고장 원인을 단계적으로 분석하고, 작은 부품 하나까지 세밀하게 관찰하며 문제를 해결했던 경험은, 코드 작성과 디버깅 과정에서 논리적으로 문제를 파악하고 효율적인 해결책을 찾아내는 능력으로 자연스럽게 이어집니다. 이러한 경험을 바탕으로 프론트엔드 개발자로 성장 중이며, ESTsoft 부트캠프를 통해 기초를 탄탄히 다지면서도 단순히 기능을 구현하는 데 그치지 않고, 각 기능의 원리와 동작 방식을 깊이 이해하려는 자세를 유지하고 있습니다. 이를 통해 사용자 중심의 직관적이고 효율적인 웹 서비스를 만들 수 있는 개발자가 되고 싶습니다.

경험

웹사이트 클론 제작 연습

SSG, NETFLEX 등 실제 서비스 UI를 분석하고 일부 페이지를 클론 코딩했습니다. 클론코딩 과정에서 구조 파악, 컴포넌트화, 재사용 가능한 CSS 구조 설계를 연습하며 효율적인 코드 작성 방법을 연습했습니다. 또한 시멘틱 태그를 적극 활용하여 코드의 가독성과 유지보수성을 높이는 방법을 학습했습니다.

부트캠프 기초 과제 수행

ESTsoft 프론트엔드 부트캠프를 참여하면서 HTML, CSS 이론을 마쳤고 Flex와 Grid를 활용하여 카드 UI 구성 및 레이아웃 정리를 수행하였습니다. 현재는 개인 프로젝트로 이력서 페이지 제작을 하고 있으며 실제 화면 개발 경험을 확장하고 있습니다. 또는 모바일, 태블릿, PC에 따른 반응형 CSS을 구성함으로서 화면 구성 능력을 기르고 있습니다.
