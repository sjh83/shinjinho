웹사이트 클론 제작 연습

Youtube, NETFLEX 등 실제 서비스 UI를 분석하고 일부 페이지를 클론 코딩했습니다. 클론코딩 과정에서 구조 파악, 컴포넌트화, 재사용 가능한 CSS 구조 설계를 연습하며 효율적인 코드 작성 방법을 연습했습니다.

부트캠프 기초 과제 수행

ESTsoft 프론트엔드 부트캠프를 참여하면서 HTML, CSS 이론을 마쳤고 Flex와 Grid를 활용하여 카드 UI 구성 및 레이아웃 정리를 수행하였습니다. 현재는 개인 프로젝트로 이력서 페이지 제작을 하고 있으며 실제 화면 개발 경험을 확장하고 있습니다. 또는 모바일, 태블릿, PC에 따른 반응형 CSS을 구성함으로서 화면 구성 능력을 기르고 있습니다.

강점

문제가 바로 해결되지 않아도 끝까지 시도하는 끈기를 바탕으로, UI 동작과 화면 흐름을 꼼꼼히 점검하며 문제를 해결합니다.
발견한이슈는 임시 처리에 그치지 않고, 의도한 동작이 안정적으로 구현될 때까지 책임감을 가지고 마무리합니다.
